# 기여 가이드

## 프로젝트 목표

exp-message는 여러 기술 스택을 활용하여 구축하는 **아주 튼튼한 채팅 서비스 제작** 프로젝트입니다.

- **백엔드**: 안정성, 확장성, 성능을 갖춘 엔터프라이즈급 채팅 서버
- **프론트엔드**: 다른 웹에 쉽게 삽입 가능한 라이브러리 형태

## 우선순위 결정 방식

- **1순위**: 핵심 기능 구현 (피드백 창구, 참여형 콘텐츠 창구)
- **2순위**: 대기업 취업에 도움이 될 만한 하고 싶은 것

## 주요 기능 개발 진행상황

### 개발 완료
- 사용자 인증/인가 (JWT) 
- 채팅방 관리 (생성, 수정, 삭제, 멤버 관리)
- 메시지 전송/수신, 실시간 통신 (WebSocket)

### 개발 예정
- 백엔드 도메인 확장 
  - 기존 "채팅방"을 "고객-쓰레드"등으로 개선 필요
- 알림 시스템 연동
  - 메시지 전송 시 실시간으로 Noti할 수 있는 수단 필요
- 프론트엔드 (라이브러리) 개발
  - 다양한 서비스에 쉽게 넣을 수 있도록 플로팅 버튼 형태 등으로 제공 필요
- 메시지 검색 (ElasticSearch 고려 중)
  - 유저들의 채팅 내역을 조회할 수 있는 검색 기능 필요
- 참여형 콘텐츠
  - 유저들의 활동을 높일 수 있는 참여형 콘텐츠 기능 필요

## 작업 진행상황 공유 방법

작업 중인 내용은 **작업중** 태그를 달아둔 PR로 공유합니다. 진행 상황을 투명하게 공개하여 협업 기반을 마련합니다.
