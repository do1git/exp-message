# exp-message

## 프로젝트 소개

exp-message는 여러 기술 스택을 활용하여 구축하는 **아주 튼튼한 채팅 서버** 프로젝트입니다.

이 프로젝트의 목적은 안정성, 확장성, 성능을 모두 갖춘 엔터프라이즈급 채팅 서버를 개발하는 것입니다.
프론트엔드는 최소한의 기능만 구현하여 백엔드 서버의 안정성과 성능에 집중합니다.

## 주요 기능

- **사용자 인증/인가**: JWT 기반 사용자 인증 및 권한 관리
- **채팅방 관리**: 채팅방 생성, 수정, 삭제 및 멤버 관리
- **메시지 전송/수신**: 실시간 메시지 전송 및 수신
- **메시지 검색**: Elasticsearch를 활용한 메시지 검색
- **실시간 통신**: WebSocket을 통한 실시간 메시지 전송
- **푸시 알림**: 이벤트 기반 푸시 알림
