# 패치노트

이 디렉토리는 프로젝트의 데일리 패치노트를 기록하는 공간입니다.

## 작성 방법

각 날짜별로 패치노트를 작성합니다. 버전이 추가된 날에는 파일명에 버전을 함께 표기합니다.

### 파일 명명 규칙

- 기본 형식: `YYYY-MM-DD-번호-설명.md` (예: `2026-01-06-01-프로젝트초기설정.md`, `2026-01-06-02-기술스택선정.md`)
- 버전 추가 형식: `YYYY-MM-DD-번호-Release-v0.0.1.md` (예: `2026-01-06-01-Release-v0.0.1.md`)

같은 날 여러 패치노트를 작성할 경우 번호를 증가시켜 구분합니다. 파일명에 간단한 설명을 포함하여 내용을 쉽게 파악할 수 있도록 합니다. 버전이 릴리즈된 날에만 버전을 파일명 뒤에 추가합니다.

### 패치노트 템플릿

```markdown
# 패치노트 - YYYY-MM-DD

## 목표했던 내용 1
- 

## 변경사항 1
- 

## 목표했던 내용 2
- 

## 변경사항 2
- 
```

각 목표와 그에 대응하는 변경사항을 1:1로 매핑하여 인과관계를 명확히 합니다.
